Notes:
Internet Explorer 10 and above:
Does not support conditional comments unless rendered with a lower document mode less than or equal to IE 9.
Internet Explorer Mobile 11:
Will render some wekbit properites due to changes to the browser in Windows Phone 8.1 GDR1
Google Chrome:
From version 28, Google Chrome uses a forked version of Webkit (Blink), however Google Chrome will still report as true to the webkit engine test.
HiDPI Devices:
The HiDPI device test relies on CSS3 support, so if CSS3 support is not available then the test will return false
